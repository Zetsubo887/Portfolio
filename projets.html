Prédiction de Salaires - Offres d'Emploi APEC

Python Machine Learning Web Scraping Dash / Web App

Description : Implémentation d'une chaîne de traitement complète (End-to-End) visant à estimer la fourchette salariale d'une offre d'emploi à partir de ses caractéristiques textuelles et structurelles.

Objectif & Problématique

Réduire l'opacité salariale du marché en fournissant une estimation fiable pour les offres "selon profil". Cet outil aide les jeunes diplômés à connaître leur valeur réelle et à préparer leurs négociations.

🛠 Méthodologie

  • Acquisition : Scraping via API APEC (tokens & requêtes JSON).
  • Preprocessing : Nettoyage, standardisation et passage au logarithme du salaire.
  • Modélisation : Benchmark de modèles (Random Forest, XGBoost, LM) via Rich.

Conclusion & Performance

Le Modèle Linéaire a été retenu avec un R² de 0.53. Ce choix privilégie une performance généralisable et interprétable, évitant l'overfitting des modèles plus complexes. Déploiement via Dash.

Analyse des Biais Culturels des IA (Compar:IA)

Python Streamlit Analyse de Biais NLP / TF-IDF

Description : Dashboard interactif analysant les usages et les biais de représentation des modèles de LLM, basé sur les données de la plateforme Compar:IA (beta.gouv.fr).

Objectif & Problématique

Évaluer si les IA reproduisent des biais de représentation culturelle sur le patrimoine français ou si certains modèles sont plus équilibrés sur les données nationales.

🛠 Méthodologie

  • Open Data : Croisement avec les bases Joconde (988k œuvres) et Basilic (86k équipements).
  • Analyse : Calcul de scores de biais via TF-IDF et analyse de fréquences.
  • Architecture : Structure modulaire avec tests unitaires (Pytest).

Conclusion & Apports

Le cockpit permet de comparer les winrates par modèle jusqu'en février 2026. L'analyse révèle des disparités significatives dans la représentation des domaines artistiques français.

Décisions Promotionnelles dans la Grande Distribution

R Économétrie de Panel Modèles Logit

Description : Étude économétrique visant à modéliser les déterminants stratégiques du déclenchement des promotions sur un panel de 534 marques de supermarchés.

Objectif & Problématique

Analyser comment les stocks et les ventes passées influencent la probabilité de mise en promotion. L'étude cherche à savoir si la promotion est un outil de gestion des invendus ou de discrimination par les prix.

🛠 Méthodologie

  • Données : Analyse de panel sur 29 mois.
  • Modélisation : Estimations par modèles Logit à Effets Fixes et Aléatoires (Arbitrage par test de Mundlak).
  • Dynamique : Utilisation d'un modèle dynamique pour capter la persistance temporelle des promotions.

Conclusion & Résultats

L'étude confirme qu'un stock élevé augmente la probabilité de promotion. On observe également une forte persistance : une promotion à t-1 rend très probable une promotion à t, surtout pour les marques nationales.
